# rke-ansible
THIS IS ONLY MADE FOR UBUNTU and LOCALHOST!
## Prerequests
Clone this repository
```
git clone https://github.com/grischdian.de/rke-ansible.git
```
Install ansbible
```
apt install ansible
```
Run Ansible
```
cd rke-ansible
ansible-playbook kubernetes.yaml
```
Check Kubernetes
```
kubectl get nodes
```
